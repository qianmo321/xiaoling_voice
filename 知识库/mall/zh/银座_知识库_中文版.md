# 银座（Ginza）购物知识库 · 中文版

> **一句话简介**：银座位于东京都中央区，是日本最具代表性的**高端商业区**，被誉为「东京的心脏」「步行者的天堂」，与纽约第五大道、伦敦牛津街齐名，是**世界三大名街**之一，也以**日本地价最高的地段**闻名。
>
> **核心地标**：以「**银座四丁目十字路口**」为中心，主干道「**中央通（银座通）**」贯穿银座一丁目至八丁目，两侧汇聚老字号百货、国际一线品牌旗舰店、米其林餐厅与百年老铺。

## 2. 历史与名字由来

「银座」这一名称由**江户幕府**命名，源自当时负责**收购与管理银币**的官方组织。在江户时代，特定的工商组织被称为「座」；银座二丁目一带曾设有管理银货的机构「银座」的事务所与银币铸造所，地名由此而来。

银座毗邻起始于日本桥的**东海道**（连接江户与京都的经济物流大动脉），因此虽以铸造师聚居的街道起家，主街两侧却很早就商店林立。这份繁华历经明治、昭和、平成，一直延续至今。

代表性老字号中，**三越百货**的前身是 1673 年创业的和服店「越后屋」，银座店于 1930 年开业，90 余年来一直是银座的门面之一。

## 3. 主要百货与商业设施

银座的购物核心是**中央通（银座通）**，两侧集中了老字号百货与国际品牌旗舰店。以下按类型梳理主要设施。

### 3.1 老字号百货

| 设施 | 亮点 |
|------|------|
| **银座三越** | 银座代表性老字号百货，位于四丁目十字路口。前身为 1673 年的「越后屋」和服店，1930 年开设银座店。入口的**狮子铜像**是著名合影点；美食楼层（地下）人气极高，新馆内还设有以金鱼为主题的「艺术水族馆美术馆」。 |
| **松屋银座** | 创业于 1881 年，是日本最古老的百货之一，以**法式优雅**风格著称。馆内的「LV 松屋银座店」外墙采用日式雅致的「江户小纹」纹样。 |
| **和光（SEIKO HOUSE GINZA）** | 四丁目十字路口的**钟楼地标**，日剧电影常见的银座象征，代表日本上流商圈。（注：建筑于 2022 年更名为 SEIKO HOUSE GINZA，但仍常被称作「和光」。） |
| **松坂屋银座店（旧）** | 已于原址上重建为下述的 GINZA SIX。 |

### 3.2 大型复合商业设施

| 设施 | 亮点 |
|------|------|
| **GINZA SIX** | 银座**规模最大**的商业设施，2017 年开业，由建筑事务所赫尔佐格与德梅隆设计，建于松坂屋银座店旧址。地下 2 层至地上 6 层共 **200 余家**店铺（含多个日本首店），涵盖 LV、Gucci 等大牌。**6F 茑屋书店**艺术藏书丰富；顶楼有银座最大的**屋顶庭园**「GINZA SIX 花园」；地下 3 层有能乐剧场「观世能乐堂」。营业约 10:30–20:30（餐厅至 23:30）。地址：中央区银座 6-10-1，银座站 A3 出口步行约 2 分钟。 |
| **东急 PLAZA 银座（TOKYU PLAZA GINZA）** | 位于数寄屋桥交叉口的地标，外观以日本传统工艺「**江户切子**」为灵感、以「光之容器」为设计概念。走时尚流行尖端，6~7 楼有日本职人工艺与文艺杂货区，并设有东京最大的机场型免税店。营业约 11:00–21:00，银座站 C2/C3 出口步行约 1 分钟。 |
| **丸井有乐町（marui）** | 位于 JR 有乐町站对面，地理位置优越，风格品牌与艺术活动空间兼具。 |
| **阪急 MEN'S TOKYO** | 距 JR 有乐町站步行约 1 分钟，1~8 层几乎全为**男性服饰**，主题为「为活跃于世界舞台的男士」。 |

### 3.3 老字号专门店

| 店铺 | 亮点 |
|------|------|
| **伊东屋（G.Itoya）** | 银座代表性的时尚**文具专卖店**，商品极其丰富，从基础文具到办公家用小物一应俱全。 |
| **鸠居堂** | 创始于 1663 年的**文房四宝**百年老店，曾为皇宫御用；笔墨纸砚与和纸工艺品齐全，适合选购精美礼品。 |
| **博品馆 TOY PARK** | 位于八丁目，号称日本最大的**玩具百货**，从公仔、布偶到电子游戏、魔术道具应有尽有。 |
| **山野乐器 银座本店** | 乐器、音乐 CD 与乐谱的老铺，7 楼常办艺人活动与讲座。 |

## 4. 文化艺术景点

银座不只购物，也是东京重要的文化艺术聚集地。

- **歌舞伎座**：日本传统表演艺术「歌舞伎」的代表性专门剧场，位于银座四丁目。现建筑为融合古典与现代的复合设施（歌舞伎座＋办公塔），设 1800 多个座位，每月更换上演剧目，提供**英文导览耳机**。塔楼 5 楼设有「歌舞伎座展廊」，展示实际使用的戏服道具，另有屋上庭园。地址：中央区银座 4-12-15，东银座站旁、银座站步行约 5 分钟。
- **Ginza Sony Park**：以多层文化设施呈现科技与艺术互动展览，定期举办工作坊与现场演出。
- **SHISEIDO THE GINZA（资生堂）**：资生堂的旗舰美妆体验空间，各楼层有不同主题的化妆品卖场与美容咨询。
- **三菱一号馆美术馆**（近日比谷）：红砖文艺复兴风格建筑，专注明治至昭和初期的西洋美术收藏，庭园宜人。

## 5. 步行者天国（歩行者天国）

银座主干道「中央通」在**周六日及节假日**实施步行者天国，从八丁目（新桥侧）到一丁目全段**禁止车辆通行**，行人可自由漫步、以品牌旗舰店为背景拍照。

- 通常于每周六、日及国定假日实施，**可能因天候中止**；年末年初的实施时间不固定。
- 这一「车辆禁行日购物散步」已成为银座的一种传统。

## 6. 美食推荐

银座是日本屈指可数的美食天堂，米其林餐厅云集，也有平价名店。

| 类型 | 推荐 |
|------|------|
| 高级寿司 | **久兵卫（银座）**——银座顶级寿司名店之一。 |
| 平价寿司 | **美登利寿司**——高鲜度、价格亲民，招牌为大鳗鱼握寿司，常需排队。 |
| 洋食/甜点 | **资生堂 Parlour（Shiseido Parlour）**——历史悠久的洋食与甜点名店。 |
| 咖啡老铺 | **银座 café paulista**——历史悠久的喫茶店。 |
| 面包老铺 | **木村家（Kimuraya）**——1874 年发明红豆面包（あんぱん）的老字号。 |
| 乌龙面 | **TsuruTonTan（つるとんたん）**——人气乌龙面店，面碗超大。 |
| 海鲜 | 近邻**筑地场外市场**，新鲜寿司、海鲜丼、天丼众多（多数店午后休息，建议早去）。 |

## 7. 交通指南

银座地铁交通极为便利，从新宿、涩谷、上野等主要枢纽多在 30 分钟内可达。

| 车站 | 线路 |
|------|------|
| **银座站** | 东京 Metro 银座线、日比谷线、丸之内线（三线交汇） |
| **有乐町站** | JR、东京 Metro 有乐町线（银座西侧入口） |
| **银座一丁目站** | 东京 Metro 有乐町线 |
| **东银座站** | 都营浅草线、东京 Metro 日比谷线（歌舞伎座旁） |
| **新桥站** | 多线交汇（银座八丁目方向） |

- 从**东京站**搭东京 Metro 丸之内线到银座站仅约 2 分钟；步行约 15 分钟也可抵达银座通。
- **都营浅草线**可从东银座方向直达浅草方向，方便串联行程。

## 8. 免税 · 退税与购物贴士

- **免税/退税**：大型百货与商场基本支持免税购物，**出示护照**即可办理；部分提供中文导购。
- **营业时间**：一般商店多为 10:00–20:00 前后，百货通常比一般商店提早约 1 小时（傍晚 19~20 点）关门；**晚上较难在银座逛商场采购**，建议白天安排购物。
- **支付**：日本消费需付消费税，退税需在指定柜台当场办理，事后一般无法补办（以各商场规定为准）。
- **议价**：百货与专门店基本不议价，也无需给小费。
- **拍照名场面**：四丁目的**和光钟楼**、三越门口的**狮子铜像**是经典合影点。
