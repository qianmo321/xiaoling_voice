# 銀座（Ginza）ショッピング・ナレッジベース · 日本語版

> **一言紹介**：銀座は東京都中央区に位置する、日本を代表する**高級商業エリア**。「東京の心臓」「歩行者の天国」と称され、ニューヨークの五番街、ロンドンのオックスフォード・ストリートと並ぶ**世界三大名街**の一つ。**日本一地価の高い地域**としても知られる。
>
> **中心ランドマーク**：「**銀座四丁目交差点**」を中心に、メインストリート「**中央通り（銀座通り）**」が銀座一丁目から八丁目まで貫き、両側に老舗百貨店・国際一流ブランドの旗艦店・ミシュランレストラン・百年老舗が集まる。

## 2. 歴史と地名の由来

「銀座」という名称は**江戸幕府**によって名付けられ、当時**銀貨の買取・管理**を行った公的組織に由来する。江戸時代、特定の商工組織は「座」と呼ばれ、銀座二丁目一帯には銀貨を管理する「銀座」の役所と銀貨鋳造所が置かれていた。地名はここから来ている。

銀座は日本橋を起点とする**東海道**（江戸と京都を結ぶ経済・物流の大動脈）に隣接していたため、鋳造師が暮らす街から始まりながらも、早くからメイン通りの両側に商店が立ち並んだ。この繁栄は明治・昭和・平成を経て現在まで続く。

代表的な老舗のうち、**三越**の前身は1673年創業の呉服店「越後屋」で、銀座店は1930年に開業。以来90年余り、銀座の顔の一つであり続けている。

## 3. 主要百貨店・商業施設

銀座のショッピングの核は**中央通り（銀座通り）**で、両側に老舗百貨店と国際ブランドの旗艦店が集中する。以下、タイプ別に主要施設を整理。

### 3.1 老舗百貨店

| 施設 | 見どころ |
|------|----------|
| **銀座三越** | 銀座を代表する老舗百貨店。四丁目交差点に位置。前身は1673年の呉服店「越後屋」、1930年に銀座店を開設。入口の**ライオン像**は有名な待ち合わせ・撮影スポット。地下の食品フロアは大人気で、新館には金魚をテーマにした「アートアクアリウム美術館」も。 |
| **松屋銀座** | 1881年創業、日本有数の老舗百貨店で、**フレンチシック**な雰囲気で知られる。館内の「ルイ・ヴィトン松屋銀座店」の外壁には和の意匠「江戸小紋」が用いられている。 |
| **和光（SEIKO HOUSE GINZA）** | 四丁目交差点の**時計塔のランドマーク**。ドラマや映画でおなじみの銀座の象徴で、日本の上流商圏を代表する。（注：2022年に建物名が SEIKO HOUSE GINZA へ改称されたが、今も「和光」と呼ばれることが多い。） |
| **松坂屋銀座店（旧）** | 跡地に後述の GINZA SIX が建てられた。 |

### 3.2 大型複合商業施設

| 施設 | 見どころ |
|------|----------|
| **GINZA SIX** | 銀座で**最大規模**の商業施設。2017年開業、設計はヘルツォーク&ド・ムーロン、松坂屋銀座店跡地に建設。地下2階~地上6階に**200以上**の店舗（日本初出店も多数）が入り、ルイ・ヴィトンやグッチなどが揃う。**6階の蔦屋書店**はアート関連の蔵書が豊富。屋上には銀座最大の**屋上庭園**「GINZA SIX ガーデン」、地下3階には能楽堂「観世能楽堂」。営業はおよそ10:30–20:30（レストラン23:30まで）。住所：中央区銀座6-10-1、銀座駅A3出口より徒歩約2分。 |
| **東急プラザ銀座（TOKYU PLAZA GINZA）** | 数寄屋橋交差点のランドマーク。外観は日本の伝統工芸「**江戸切子**」に着想を得た「光の器」がコンセプト。流行の最先端をゆき、6~7階には日本の職人工芸・文芸雑貨のフロア、東京最大級の空港型免税店も。営業はおよそ11:00–21:00、銀座駅C2/C3出口より徒歩約1分。 |
| **丸井有楽町（マルイ）** | JR有楽町駅の向かいという好立地。個性的なブランドとアートイベントスペースを併設。 |
| **阪急MEN'S TOKYO** | JR有楽町駅より徒歩約1分。1~8階のほぼ全フロアが**紳士服**で、「世界の舞台で活躍する男性のために」がテーマ。 |

### 3.3 老舗専門店

| 店舗 | 見どころ |
|------|----------|
| **伊東屋（G.Itoya）** | 銀座を代表するおしゃれな**文具専門店**。基礎的な文房具からオフィス・家庭用の便利グッズまで品揃えが非常に豊富。 |
| **鳩居堂** | 1663年創業の**文房四宝**（筆・墨・紙・硯）の老舗。皇室御用達の歴史を持ち、和紙工芸品も揃い、贈答品選びに好適。 |
| **博品館 TOY PARK** | 八丁目にある、日本最大級の**玩具百貨店**。フィギュア・ぬいぐるみからテレビゲーム・手品用具まで何でも揃う。 |
| **山野楽器 銀座本店** | 楽器・音楽CD・楽譜の老舗。7階では often アーティストのイベントや講座を開催。 |

## 4. 文化・アートスポット

銀座はショッピングだけでなく、東京有数の文化・アートの集積地でもある。

- **歌舞伎座**：日本の伝統芸能「歌舞伎」の代表的専門劇場で、銀座四丁目に位置。現在の建物は古典と現代を融合した複合施設（歌舞伎座＋オフィスタワー）で、1,800以上の座席を備え、月替わりで演目を上演。**英語の音声ガイド**も利用可能。タワー5階には「歌舞伎座ギャラリー」があり、実際に舞台で使われた衣裳・小道具を展示、屋上庭園もある。住所：中央区銀座4-12-15、東銀座駅すぐ・銀座駅より徒歩約5分。
- **Ginza Sony Park**：多層の文化施設で、テクノロジーとアートのインタラクティブな展示を展開。ワークショップやライブも定期開催。
- **SHISEIDO THE GINZA（資生堂）**：資生堂の旗艦ビューティー体験空間。各フロアに異なるコンセプトの化粧品売場とビューティーカウンセリング。
- **三菱一号館美術館**（日比谷近く）：赤煉瓦のルネサンス様式建築。明治~昭和初期の西洋美術コレクションに注力し、庭園も心地よい。

## 5. 歩行者天国

銀座のメインストリート「中央通り」は、**土日祝**に歩行者天国を実施し、八丁目（新橋側）から一丁目まで全区間で**車両通行止め**となる。歩行者は自由に歩き、ブランド旗艦店を背景に撮影を楽しめる。

- 通常は毎週土・日および国民の祝日に実施されるが、**天候により中止**の場合がある。年末年始の実施時間は不定。
- この「車両通行止めの日にショッピング散歩」は、銀座の一つの伝統となっている。

## 7. アクセス

銀座は地下鉄アクセスが非常に便利で、新宿・渋谷・上野などの主要ターミナルから30分以内で到着できることが多い。

| 駅 | 路線 |
|----|------|
| **銀座駅** | 東京メトロ銀座線・日比谷線・丸ノ内線（三線交差） |
| **有楽町駅** | JR、東京メトロ有楽町線（銀座の西側入口） |
| **銀座一丁目駅** | 東京メトロ有楽町線 |
| **東銀座駅** | 都営浅草線、東京メトロ日比谷線（歌舞伎座すぐ） |
| **新橋駅** | 複数路線が交差（銀座八丁目方面） |

- **東京駅**から東京メトロ丸ノ内線で銀座駅まで約2分。徒歩約15分でも銀座通りに到達可能。
- **都営浅草線**は東銀座方面から浅草方面へ直通し、行程の連結に便利。

## 8. 免税・還付とショッピングのヒント

- **免税/税還付**：大型百貨店・商業施設は基本的に免税ショッピングに対応。**パスポート提示**で手続き可能。一部は多言語対応。
- **営業時間**：一般店舗は10:00–20:00前後が多く、百貨店は一般店より約1時間早く（夕方19~20時頃）閉まる傾向。**夜は銀座で買い物しづらい**ため、ショッピングは日中に組むのがおすすめ。
- **支払い**：日本での買い物には消費税がかかり、還付は指定カウンターでその場で手続きする必要がある（後日不可の場合が多い。各施設の規定に従う）。
- **値引き**：百貨店・専門店では基本的に値引きはなく、チップも不要。
- **撮影の定番**：四丁目の**和光の時計塔**、三越入口の**ライオン像**が定番の撮影スポット。
